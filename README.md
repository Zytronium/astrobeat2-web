# Astrobeat 2

A mobile and web music player based on my original Astrobeat.

## Web:
- Password-protected page for uploading videos to online storage
- API for downloading music files
- Page for creating and organizing playlists
- Page for listening to music (stretch goal)

## Mobile:
- Listen to music
- Downloads songs when you play them, retains on phone for 3 months after last time played
- View playlists
- Play single music file
- Auto generate playlist from genre or tags (stretch goal)
- Upload music files (stretch goal)

## Storage:
- Neon DB for metadata
- Cloudflare R2 for music file hosting
- Monthly cost: **$0**
